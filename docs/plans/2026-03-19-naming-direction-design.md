# 命名方向测试 PRD

**一句话定义：** 用一套可分享的“命名方向测试”先帮准父母找到共识，再基于结果进入起名。

## 1. 产品定位

- **目标用户：** 25-35 岁准父母，优先是年轻父母本人，次级用户是伴侣与会参与命名决策的家人。
- **核心场景：** 用户先测“更偏大雅还是大俗”，拿到结果后发给伴侣或家人讨论，再沿着方向继续起名。
- **核心价值：**
  - 降低“先生成一堆名字但方向没对齐”的决策摩擦
  - 用文化感替代玄学感，避免掉进传统起名网站的表达套路
  - 让结果本身成为可晒、可讨论、可继续行动的内容

## 2. 差异点

- **不是直接给名字，而是先给方向。**
  依据：用户真实分歧往往发生在“偏什么路子”而不是“少几个候选名”。
- **不是拼玄学，而是拼解释力和传播力。**
  依据：年轻父母反感五行八字式说教，但愿意接受有画面感、有审美判断的分类。
- **不是单人工具，而是社交决策物。**
  依据：结果页要能发给伴侣或家人，引发讨论并推动下一步行动。

## 3. 分类体系

- **前台主轴：** `大雅 / 大俗`
- **测试原型：**
  - `雅正型`
  - `空明型`
  - `天真型`
- **文化解释层：**
  - `偏儒`
  - `偏佛`
  - `偏道`
- **辅助描述：** `中正感` 不做主类，只用于结果页中的“家人接受度”说明。

### 类型定义

| 阵营 | 原型 | 文化底色 | 关键词 | 示例名 |
|---|---|---|---|---|
| 大雅 | 雅正型 | 偏儒 | 端正、耐看、克制、书卷气 | 知言 / 修远 / 见初 |
| 大雅 | 空明型 | 偏佛 | 留白、清净、轻盈、有余味 | 清和 / 闻溪 / 见山 |
| 大俗 | 天真型 | 偏道 | 山野、自然、生机、人间感 | 春生 / 大川 / 田野 |

## 4. 页面与信息架构

```text
首页
├── 主入口：开始测试
├── 次入口：直接起名
├── 结果预览卡（3 个原型）
└── 场景说明与信任文案

测试页
└── 5 题单题单屏问答

测试结果页
├── 阵营标签（大雅 / 大俗）
├── 原型结果（雅正 / 空明 / 天真）
├── 一句话判断
├── 示例名
├── 家人接受度说明
├── 分享 CTA
└── 起名 CTA

分享落地页
├── TA 的结果卡
├── 我也测一下
└── 按 TA 的方向起名

起名页
├── 当前方向标签
├── 姓氏 / 性别 / 避用字
└── 沿该方向生成候选名
```

## 5. 用户流程

### Happy Path

1. 用户从首页进入测试
2. 30-45 秒完成 5 题
3. 结果页先看到 `大雅/大俗 + 原型 + 一句话判断`
4. 看到 3 个示例名，确认结果不是空话
5. 发给伴侣或家人
6. 对方打开落地页后继续测试或直接起名
7. 用户按当前方向进入起名页生成候选名

### 关键异常路径

- **不认同结果：** 提供“重测/重新开始”，避免把错误结果硬塞给用户
- **看到大俗有心理负担：** 对外展示使用 `偏大俗 · 天真型`
- **只想直接起名：** 首页保留 `直接起名` 次入口
- **家人不理解结果：** 在结果页中补“家人接受度 / 稳妥度”说明

## 6. 首页文案基线

### Hero

- **主标题：** 测你家起名更偏大雅还是大俗
- **副标题：** 不用生辰八字，先找到你们更合拍的命名方向
- **主按钮：** 开始测试
- **次按钮：** 直接起名

### 结果预览卡

- `偏大雅 · 雅正型`
  - 端正耐看，越念越有味道
  - 知言 / 修远 / 见初
- `偏大雅 · 空明型`
  - 清净留白，轻盈但不寡淡
  - 清和 / 闻溪 / 见山
- `偏大俗 · 天真型`
  - 有生机，有人间感，不故作高深
  - 春生 / 大川 / 田野

### 解释区

- **标题：** 为什么先测，再起名？
- **文案：** 很多人不是缺名字，而是还没想清楚到底喜欢哪一路。

## 7. 测试题文案基线

1. 如果把名字想成一幅画，你更喜欢哪种感觉？
   - 灯下翻书，安静端正
   - 雨后空山，轻淡有余味
   - 风过田野，自然有生气

2. 下面三组名字，你第一眼更偏向哪组？
   - 知言 / 修远 / 见初
   - 清和 / 闻溪 / 见山
   - 春生 / 大川 / 田野

3. 你最不想要哪种名字？
   - 太文绉绉，像故意显得有文化
   - 太轻太淡，听完记不住
   - 太直白，像一眼看完没余味

4. 给宝宝起名时，你更在意什么？
   - 我一眼就喜欢
   - 有味道，但不要太难懂
   - 长辈一听顺耳，也容易接受

5. 你更希望孩子长大后，这个名字给人的感觉是？
   - 端正、克制、可靠
   - 清净、轻盈、有余味
   - 生动、自然、有生命力

## 8. 结果页文案骨架

### 雅正型

- 你家起名更偏大雅
- 雅正型
- 你们更容易被那些端正、耐看、不过分张扬的名字打动。
- 示例名：知言 / 修远 / 见初
- 家人接受度：常用字更多，读起来更稳，更顺耳

### 空明型

- 你家起名更偏大雅
- 空明型
- 你们喜欢的名字，多半不靠堆砌寓意取胜，而是轻一点、静一点，留一点余味。
- 示例名：清和 / 闻溪 / 见山
- 家人接受度：清爽克制，需要一句话解释其美感

### 天真型

- 你家起名更偏大俗
- 天真型
- 你们更喜欢有生机、有人间感的名字，不故作高深。
- 示例名：春生 / 大川 / 田野
- 家人接受度：记忆点强，建议强调常用字与自然感，避免“土气”解读

## 9. 视觉方向

- **基调：** 新文人 + 内容测试感
- **关键词：** 留白、纸感、低饱和、结果海报感
- **避免：** 八卦、罗盘、深黑金色、符咒感、过重宗教符号

## 10. 成功标准

- 用户能在 5 秒内理解首页主张
- 用户能在 45 秒内完成测试
- 结果页既能截图分享，也能顺畅进入起名页
- 起名页能明确承接“当前方向”，而不是回到无差别输入
